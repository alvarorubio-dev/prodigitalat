# ProDigital 🚀

Plataforma web avanzada diseñada para impulsar negocios de servicios en Estados Unidos mediante soluciones digitales de alto rendimiento. Desarrollada bajo la visión de Sormenak, enfocada en la automatización y soluciones impulsadas por IA.

## 🛠️ Stack Tecnológico

Este proyecto está construido con herramientas de nueva generación para garantizar la máxima velocidad y escalabilidad:

* **Framework:** [React 19](https://react.dev/)
* **Build Tool:** [Vite 8](https://vite.dev/) (Optimizado con motor OXC para ultra-alto rendimiento)
* **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)

## ⚙️ Requisitos Previos

Asegúrate de tener instalado en tu entorno local:
* [Node.js](https://nodejs.org/) (Versión 18 o superior recomendada)
* npm (o pnpm/yarn)
